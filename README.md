# groupie-trackerg18
Ce projet consiste à développer une application GUI en Go avec le framework Fyne, capable de consommer une API complexe pour visualiser les données de groupes de musique via des interfaces ergonomiques respectant les règles de Schneiderman. L'application intègre un système de géolocalisation convertissant les adresses en coordonnées pour afficher les concerts sur des cartes interactives, une barre de recherche intelligente avec suggestions dynamiques (par nom, membre, lieu, etc.), ainsi que des filtres avancés (plage de dates et cases à cocher) pour affiner les résultats. Enfin, l'expérience utilisateur est enrichie par des fonctionnalités bonus telles que la gestion d'un système de favoris et l'intégration de liens de streaming vers des plateformes comme Spotify ou YouTube.
